# Einmal ohne, bitte

Nextjs graphcms project.

## Install

`yarn`

## Develop

- Add `.env` file including

```
GQL_CMS_TOKEN=...
GQL_CMS_ID=...
EMAIL=...
MAILGUN_API=...
NEXT_PUBLIC_MATOMO_URL=...
```

- run `yarn dev`

## Graphql Codegen

- run `yarn codegen`

## Build

- run `yarn build`

## Tests

- run `yarn test`.
